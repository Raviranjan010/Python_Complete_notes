# Functions Introduction

A function is a block of code which only runs when it is called.
It allows us to **Reuse Code** (Don't Repeat Yourself - DRY principle).

## Syntax
```python
def function_name():
    # Code block
```

## Calling a Function
```python
function_name()
```

## Types of Functions
1.  **Built-in**: `print()`, `len()`, `type()`, `input()`.
2.  **User-Defined**: Functions created by us using `def`.
