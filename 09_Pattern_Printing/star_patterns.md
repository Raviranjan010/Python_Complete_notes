# Star Patterns

## 1. Square Pattern
```
* * *
* * *
* * *
```
Logic: Rows = 3, Cols = 3.

## 2. Right Triangle
```
*
* *
* * *
```
Logic:
-   Row 0: 1 star
-   Row 1: 2 stars
-   Row i: i+1 stars

## 3. Inverted Triangle
```
* * *
* *
*
```
Logic:
-   Row 0: 3 stars
-   Row 1: 2 stars
-   Row i: n - i stars
