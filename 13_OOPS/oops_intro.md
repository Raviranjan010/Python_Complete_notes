# OOPS Introduction

**Object-Oriented Programming (OOP)** is a programming paradigm based on the concept of "objects".
An object can contain:
1.  **Data** (Attributes/Variables)
2.  **Code** (Methods/Functions)

## Key Concepts of OOP
1.  **Class**: Examples (Blueprint)
2.  **Object**: Real-world entity
3.  **Inheritance**: Parent-Child relationship
4.  **Polymorphism**: Many forms
5.  **Encapsulation**: Hiding data
6.  **Abstraction**: Hiding implementation details
