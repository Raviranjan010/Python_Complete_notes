# If-Else Statement

The `if-else` statement allows us to execute one block if the condition is True, and another block if it is False.

## Syntax
```python
if condition:
    # Block A (True)
else:
    # Block B (False)
```

## Example
```python
num = 10
if num % 2 == 0:
    print("Even Number")
else:
    print("Odd Number")
```
